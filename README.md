## Introduction
- A template for Python-Flask project.
- This template is based on [Flask](https://flask.palletsprojects.com/en/3.0.x/)

## Usage
- Clone this repository.
```bash
git clone
```
- Install dependencies.
```bash
pip install -r requirements.txt
```
- Run the project.
```bash
python manager.py
```
- Enjoy it!
